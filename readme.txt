Convert [showmore] shortcodes into an easy to use expanding content area. 




== Description == 
This plugin convert the shortcode [showmore]  into an easy to use expanding content area. Useful for hiding images, long quotes etc.




== Installation ==


1. Unzip and upload mjshowmore.php to your plugins folder, activate and you're good to go.


2. Wrap your content in [showmore] ... [/showmore] shortcodes to get started.


3. Include a link (<a href="#">Some Link</a>)


4. The content you wish to initially hide, wrap with [more] ... [/more]
